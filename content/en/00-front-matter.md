## How to use this playbook (read this first — 90 seconds)

1. **Open `Appendix A — The Master Agent System Prompt`** at the back of this PDF. Paste it into ChatGPT or Claude *once*. That single paste trains the model in your voice for every future prompt in this book.
2. **Pick a prompt** from Section 1 by property type and price tier. Fill in the `{{variables}}`. Paste. Read the output. If anything sounds robotic, the master prompt has a self-check rule that fixes it — type *"run self-check"* and it cleans itself.
3. **Don't run two prompts in the same chat.** Each new listing = new chat. The master prompt re-loads automatically when you paste it again. Voice stays consistent.
4. **Edit the output.** Treat the AI as a junior copywriter, not a senior one. Cut adjectives. Cut filler. Add one specific number that's true.
5. **Track which prompts close.** Mark the 5–10 that get the most leads on your Bayut/Property Finder dashboards. Those are your money prompts. Re-skin them for new properties first; ignore the rest.

That's it. Everything else is optional.
