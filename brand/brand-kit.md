# Bayan — Brand Kit

## Name

**Bayan** (Arabic: بيان) — meaning "statement," "declaration," "clarity." Works bilingually: same word in both languages, recognizable to Arabic-speaking agents without translation, memorable in English without being exotic.

**Full name:** Bayan — The Listing Vault.

## Positioning line (under the name)

> *For Gulf real-estate agents who'd rather close than copywrite.*

Arabic: *للعقاريين الخليجيين الذين يفضّلون البيع على الكتابة.*

## Color palette

| Role | Name | Hex | Use |
|---|---|---|---|
| Background | Sand | `#F6F1E7` | Primary page / PDF background |
| Ink | Ink | `#141310` | Body copy, headings, primary button |
| Accent | Oud | `#3D2E1F` | Hover state, footer, serious accents |
| Highlight | Majlis | `#A06E3D` | Numbers, pull-quotes, highlight text |
| Support | Sage | `#556B54` | "Built for you" positive labels |
| Surface | Bone | `#FBF9F3` | Cards, alternating section backgrounds |

Rationale: muted desert palette that photographs well in bilingual layouts and doesn't trigger the "cheap Gumroad PDF" tell (primary-color gradients, tech-purple accents, neon CTAs).

## Typography

| Role | Font | Use |
|---|---|---|
| Headlines (EN) | Serif (ui-serif / Source Serif / similar) | H1, H2, pull-quotes |
| Body (EN) | Inter | Paragraphs, lists, buttons |
| Arabic body | Noto Naskh Arabic | All Arabic text, headlines included |
| Mono | ui-monospace | Variable placeholders (`{{name}}`), banned phrase chips |

Pairing rule: never mix three type families on the same page. Serif for headlines, Inter for body, Naskh for Arabic. No decorative display font ever.

## Wordmark

Simple type-only wordmark: `Bayan` in serif bold, letter-spacing `-0.02em`. Stand-alone, no icon. If an icon is needed (favicon, social avatar), use the Arabic letter **ب** (ba) in Ink on Sand, set inside a hairline square.

## Voice

**In 6 words:** direct, specific, numbers-first, bilingual, unshowy.

**Say:** "3 comps in the last 90 days between AED 1.55M and 1.62M"
**Don't say:** "a stunning opportunity in the heart of the marina"

**Say:** "35-day median close when priced inside band"
**Don't say:** "unlock the lifestyle of the marina"

## Forbidden brand behaviors

- Never show a cartoon avatar of an "agent" or a stock-photo Dubai skyline
- Never use the 🔥 or ✨ emoji in captions
- Never run a "limited time" countdown that isn't real
- Never claim specific dollar-value deals closed by "users of Bayan" without a documented, named testimonial
- Never auto-DM new Instagram followers (kills trust in the exact segment that would buy)
